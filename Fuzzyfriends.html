Fuzzy Friends:
A Mobile Game

Project overview

We had to take any existing hyper casual mobile game and design a similar one, whilst adding new challenges and miscellaneous features to the game. Our reference game is calledQSWaterMelon (iOS)/ QS Monkey Land: King of Fruits (Android) .
The game we developed is Fuzzy Friends. It’s all cats.

Game Design Communication Design Wireframing

Duration: 2 months Team type: Duo

Game lore

We all know how much cats love boxes. Well, that’s what the whole game is based on! Fit as many as cats in the carboard box (the main gameplay) before the owner finds them! Remember to keep them happy by giving them new looks every now and then!

Game mechanics

1. Core loop

2. Hints

If the user runs out of hints, they use the game currency (treats) to buy one hint for 50.

3. Ads integration

Ads can be watched to attain the following:

  • Double the treats after the game.
  • Unlock missed accessories/stamps.

4. Meta loop event

Daily Puzzle

  • Collect sets of 3 of the same cats.
  • The player must return the cats to their home before the timer runs out to complete the daily task.
  • Upon completion, the player either receives a stamp or an accessory, and in the case of the latter it can be customized on any cat.

Leaderboard

  • The sum of the points the player gets in each game in a fortnight is used to determine the position on the leaderboard.
  • At the end of each season, the player receives treats based on which position they finish.

Game characters and accessories

These are the cats of our game, they are the ranks as well. Accessories obtained from daily puzzles can be customized on the cats.

Seasonal Game background

There are two kinds of backgrounds, summer and winter which the user can unlock for 2000 treats each.
